# 🚀 EMPEZAR AQUÍ

¡Bienvenido! Sigue estos pasos para comenzar a usar TestJava2BBDD en **menos de 1 minuto**.

---

## 📍 Paso 1: Abre la Terminal

Abre una terminal/cmd y ve a la carpeta del proyecto

---

## ▶️ Paso 2: Ejecuta el Programa

Simplemente escribe:

```bash
./run.sh
```

**Eso es todo.** El programa:
- ✅ Compila automáticamente
- ✅ Abre el menú principal
- ✅ Está listo para usar

---

## 📋 Paso 3: Verás el Menú

Se abrirá una pantalla así:


---

## 🎯 Paso 4: Elige un Tema

Escribe el número del tema que quieres estudiar.

**Recomendado para principiantes:**
- Opción `1` - Conceptos SQL
- Opción `6` - Básicos de Java

Ejemplo: escribe `1` y presiona ENTER

```
Selecciona una opción (0-10): 1
```

---

## 👤 Paso 5: Ingresa tu Nombre

El programa te pedirá tu nombre:

```
⏳ Iniciando test...

¿Cuál es tu nombre? Juan
```

---

## 📊 Paso 6: Elige Dificultad

Elige cómo te sientes hoy:

```
¿Cómo te encuentras hoy, Juan?
a) Necesito calentarme (preguntas fáciles)
b) Voy bien (preguntas normales)
c) Dame un reto (preguntas difíciles)

Tu respuesta (a/b/c): b
```

---

## ❓ Paso 7: Responde las Preguntas

El programa te hará 30 preguntas. Para cada una:

```
P1. ¿Qué es SQL?
a) Un error
b) Lenguaje de consultas
c) Una tabla
d) Un tipo de dato

Tu respuesta (a/b/c/d): b

✅ Correcto! Bien hecho.
```

## 🔄 Paso 9: Vuelves al Menú

Presiona ENTER y vuelves al menú principal. Puedes:
- ✅ Elegir otro tema (1-10)
- ✅ Repetir el mismo test para mejorar
- ✅ Salir (opción 0)

---

## ⚠️ Si No Funciona

### Error: "Permission denied"

```bash
chmod +x run.sh
./run.sh
```

### Error: "Java not found"

Necesitas instalar Java. Descárgalo de:
https://www.oracle.com/java/technologies/downloads/

### La terminal se ve rara

Es normal. Presiona ENTER si aparece algo extraño.
